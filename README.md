# 📖 [Awesome Books] <a name="about-project"></a>

**[Awesome Books]** is a project at <a href='https://www.microverse.org/'>Microverse</a> module two.
It's a single page application that allows users to add and remove books.When a user adds a book, the book's
data is stored in local storage. This data is retrieved from local storage anytime the page loads.
